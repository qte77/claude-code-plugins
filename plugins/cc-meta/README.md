# cc-meta

Claude Code meta-skills for cross-project synthesis and session intelligence.

## Skills

- **synthesizing-cc-bigpicture** — Synthesizes a living meta-plan across Claude Code sessions, plans, tasks, and team communications. Surfaces reasoning modes (diverge/converge, inductive/deductive, top-down/bottom-up) per work stream.

## Install

```bash
claude plugin install cc-meta@qte77-claude-code-utils
```
